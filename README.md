### Репозиторий включает в себя набор автотестов для сайта https://selectel.ru

Реализованные проверки:
1. Проверка того, что несуществующая страница возвращает 404.
2. Проверка видимости логотипа на главной странице.
3. Проверка того, что главная страница открывается и заголовок содержит "Selectel"
4. Проверка навигации через меню "Продукты"
5. Проверка, что меню "Продукты" видно.

Используемые инструменты: 
- Python 3.14.3
- Pytest
- Playwrite

Как запустить проект локально:
1. Клонировать репозиторий <br>
git clone https://github.com/mihacia/qa_testovoe.git
2. Перейти в каталог qa_testovoe <br>
cd qa_testovoe
3. Создать виртуальное окружение <br>
python -m venv venv
4. Активировать его <br>
venv/Scripts/activate
5. Установить зависимости <br>
pip install -r requirements.txt
6. Запустиь тесты:<br>
pytest -v
